# Quick Start Guide

This guide will walk you through using duv'Assistant for the first time. In just 5 minutes, you'll generate your first AI-powered code completion and learn how to use the snippet library.

<Note>
Before starting, ensure you've completed the [installation](/installation) and configured your OpenAI API key.
</Note>

## Your First AI Completion

<Steps>
  <Step title="Create a New File">
    Open Visual Studio Code and create a new JavaScript file:
    
    1. Press `Ctrl+N` (Windows/Linux) or `Cmd+N` (macOS) to create a new file
    2. Save it with a `.js` extension (e.g., `test.js`)
    
    You should see the activation message:
    ```
    Extensión de autocompletado activada.
    ```
  </Step>

  <Step title="Start Writing Code">
    Begin typing a function. The extension analyzes all code from the beginning of the file to your cursor position to provide context-aware suggestions.
    
    Try this example:
    
    ```javascript
    // Function to calculate the sum of an array
    function calculateSum(arr) {
    ```
    
    As you type, duv'Assistant sends your code context to OpenAI with the prompt:
    ```
    Completa el siguiente código:
    
    [your code here]
    ```
  </Step>

  <Step title="Accept the AI Suggestion">
    When you pause typing, IntelliSense will show a completion item labeled **"Sugerencia de IA"** (AI Suggestion).
    
    - The suggestion includes code completion generated by OpenAI
    - It appears with a snippet icon in the IntelliSense menu
    - The documentation shows "Sugerencia generada por OpenAI"
    
    Press `Tab` or `Enter` to accept the suggestion.
  </Step>

  <Step title="Continue Coding">
    Keep writing code and accepting suggestions as needed. The AI provider analyzes your entire file context each time to provide relevant completions.
  </Step>
</Steps>

## Using Code Snippets

duv'Assistant includes built-in snippets for common code patterns. These work without requiring an OpenAI API call.

<Steps>
  <Step title="Create a For Loop">
    In your JavaScript or TypeScript file, type `for-loop` and wait for IntelliSense to appear.
    
    The snippet will expand to:
    
    ```javascript
    for (let i = 0; i < array.length; i++) {
        console.log(array[i])
    }
    ```
    
    <Note>
    The snippet includes placeholder navigation - press `Tab` to jump between the variable name (`i`), array name, and loop body.
    </Note>
  </Step>

  <Step title="Add Console Logging">
    Type `console-log` to insert a quick console statement:
    
    ```javascript
    console.log(mensaje);
    ```
    
    The cursor will be positioned at the `mensaje` placeholder for immediate editing.
  </Step>
</Steps>

## How the AI Provider Works

Under the hood, the AI completion provider follows this workflow:

1. **Context Gathering**: Captures all text from the start of the file to your cursor position
   ```typescript
   const textBeforeCursor = document.getText(
       new vscode.Range(new vscode.Position(0, 0), position)
   );
   ```

2. **Prompt Construction**: Wraps your code in a completion prompt
   ```typescript
   const prompt = `Completa el siguiente código:\n\n${textBeforeCursor}`;
   ```

3. **API Request**: Sends the prompt to OpenAI's API using the configured model

4. **Result Display**: Presents the suggestion in IntelliSense with the label "Sugerencia de IA"

## Understanding Error Handling

If the AI provider encounters an error (network issues, API problems, etc.), it:

1. Logs the error to the console: `Error al generar sugerencia de IA:`
2. Returns an empty completion list
3. Allows you to continue coding without interruption

Check the VS Code **Developer Console** (`Help > Toggle Developer Tools`) if you're not seeing AI suggestions.

## Multi-Language Configuration

The extension activates for languages defined in `src/config/settings.ts`:

```typescript
export const settings = {
    openAI: {
        apiKey: 'TU_API_KEY',
        model: 'code-davinci-002',
    },
    supportedLanguages: ['javascript', 'typescript', 'python'],
};
```

Both the AI provider and snippet provider are registered for each supported language during extension activation:

```typescript
settings.supportedLanguages.forEach(language => {
    const aiProvider = vscode.languages.registerCompletionItemProvider(
        { scheme: 'file', language },
        new AICompletionProvider()
    );
    
    const snippetProvider = vscode.languages.registerCompletionItemProvider(
        { scheme: 'file', language },
        new SnippetCompletionProvider()
    );
    
    context.subscriptions.push(aiProvider, snippetProvider);
});
```
