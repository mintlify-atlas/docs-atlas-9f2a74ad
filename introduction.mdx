# duv'Assistant

duv'Assistant is a Visual Studio Code extension that enhances your coding experience with AI-powered code completions and pre-built snippets. It leverages OpenAI's powerful language models to provide intelligent code suggestions based on your context, while also offering quick access to common code patterns through customizable snippets.

## What is duv'Assistant?

duv'Assistant acts as your intelligent coding companion, providing:

- **AI-Powered Completions**: Get context-aware code suggestions powered by OpenAI's API
- **Smart Snippet Library**: Access pre-built code snippets for common patterns like loops and console statements
- **Multi-Language Support**: Works seamlessly with JavaScript, TypeScript, and Python
- **Seamless Integration**: Integrates directly into VS Code's IntelliSense completion system

## Key Features

### AI Code Completion

The extension analyzes your code from the beginning of the file to your cursor position and generates intelligent completions using OpenAI's language models. Simply start typing and duv'Assistant will offer suggestions that understand your context.

### Custom Snippets

Access a growing library of built-in code snippets including:
- `for-loop` - Basic for loop structure with placeholder navigation
- `console-log` - Quick console logging statements
- More snippets for common coding patterns

### Multi-Language Support

Currently supports:
- JavaScript
- TypeScript
- Python

The extension activates automatically when working with these languages, providing completions without any manual triggers.
